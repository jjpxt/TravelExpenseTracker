**fullstack .NET 9 cross-platform mobile app using:**
**.NET MAUI + XAML +**
**ASP.NET Core Web API +**
**EF Core + SQL Server**


- Login & Registration
- Main Dashboard (Recent Trips)
- All Trips Page
- Add New Trip
- Trip Details + Expense List
- Add Expense Modal
- Edit Trip
- Profile Settings (Update Profile, Change Password)
- Manage Expense Categories
